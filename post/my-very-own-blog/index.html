My Very Own Blog

 Posted on July 13, 2024  |  1 minutes  |  205 words  |  Simon Hathout Willard

Hello!

I’ve been meaning to set up a blog for a while, to give me a nice, and convenient, way to chronicle all my different experiences. I originally tried making my own site from scratch using NeoCities to host it, as I’m a big fan of the old web aesthetics. Unfortunately, I quickly hit a wall with my HTML/CSS abilities, and although I learned a lot building it, getting my site to a presentable level would have taken dozens of hours that I simply don’t have.

Instead, at the recommendation of my friend Joshua Himmons, I set up a website using Hugo. It still took some work to get everything downloaded, but now I can simply write some markdown files, put them onto my computer, and run a couple commands to build everything into a html/css file that then gets uploaded to Github for the world to see.

Also, my icon is from the wonderful “Critters” from WetPaint, a 1986 program by Dubl-Click software, which I got from this archive

In all, this site probably took 4 hours to get up and running, which isn’t too awful. Now, my next task is to write about all the projects and fun stuff I’ve been up to!
